# Solution 6: Editable Subgrid Canvas App

## What This Solution Demonstrates

This solution demonstrates how to build a **Power Apps canvas app with an editable subgrid**, allowing users to efficiently edit multiple rows of data directly within a grid-style interface without navigating to individual record forms. This pattern is particularly useful for bulk data entry and data correction scenarios.

### Components Included

| Component | Type | Description |
|---|---|---|
| Editable Subgrid App | **Canvas App** | Canvas app featuring an inline editable grid for data management |

### Key Features

- Inline editing of multiple records within a single grid view
- Real-time field updates without requiring page navigation or form popups
- Demonstrates the `Gallery` and editable field pattern in Power Apps
- Efficient UX design for bulk data management scenarios
- Suitable as a reusable pattern for inventory management, data cleansing, and data entry apps

## Prerequisites

- A Power Platform environment (Dataverse not required for basic functionality)
- A **Premium Power Apps licence** may be required depending on how the data source is configured

## How to Import

1. Download `solution.zip` from this folder.
2. Go to [https://make.powerapps.com](https://make.powerapps.com) and select your target environment.
3. Navigate to **Solutions** → **Import solution**.
4. Browse to `solution.zip` and follow the on-screen wizard.
5. Once imported, open the canvas app from within the solution and review the data source connections.
